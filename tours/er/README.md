# 急救室 360 環景

把你的急救室全景照片存成這個檔名放在這裡：

    tours/er/resus.jpg

- 格式：**equirectangular（等距長方投影）**，寬高比 **2:1**（例如 4000×2000、2000×1000）。
- `tour.html` 的內建急救室 Demo（教案 id：`er-resus`）會讀取 `tours/er/resus.jpg`。
- 存好後 `git add tours/er/resus.jpg`，push 上 GitHub Pages 即生效。
- 熱點位置（洗手台、病人、監視器、急救車、測驗、危害）已依這張全景先擺好，可到 `tour-editor.html` 拖曳微調。

若不想放進 repo，也可以把照片上傳到任何可公開存取的網址，然後在 `tour-editor.html` 的「環景圖網址」欄位貼上完整 URL。
